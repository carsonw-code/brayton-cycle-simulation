# brayton-cycle-simulation

This is a Python-based Brayton cycle simulation with interactive sliders for exploring how different cycle parameters affect performance.

## Features
- Interactive parameter sliders
- Compressor efficiency adjustment
- Turbine efficiency adjustment
- Real-time cycle behavior visualization

## Requirements
- Python
- numpy
- matplotlib
- CoolProp

## Run
Run the file from a Python environment with the required libraries installed.
